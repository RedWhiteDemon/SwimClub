# Сценарии пользователя и администратора

- **Гость**: просматривает главную, расписание, тренеров, видео; оставляет заявку в браузере (локально).
- **Редактирование заявки**: кнопка «Изменить» под заявкой, сохранение через ту же форму.
- **API** (если запущен backend): те же сущности доступны через REST для интеграции или миграции данных с локального хранилища.
